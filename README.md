# Set-de-Live-Tiles
Voici un set de live tiles codées en C pour votre bureau Windows.
En avez vous déjà eu assé de ne pas pouvoir réstaurer les Live Tiles sur votre bureau Windows 11 ? Moi, oui.
Oubliez Live Tiles Anywhere car voici un set de Live Tiles pour votre bureau Windows !
Dans ce pack, vous trouverez : 
Une live tile XBOX Wire qui affiche dynamiquement les dernières actus Xbox
Une live tile Météo
Une live tile Actualités qui affichera toute les actus tech françaises
Une live tile Word qui affichera dynamiquement vos derniers documents Word ouverts
Téléchargez la dernière release en ZIP, extractez l'archive puis lançez toute les apps
